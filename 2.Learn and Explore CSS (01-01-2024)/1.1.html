What is CSS

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Cascading

The term "cascading" refers to the order of priority or precedence in applying styles to HTML elements. Styles can be defined at different levels, and they cascade down to determine the final appearance of an element.

Style

"Style" refers to the visual presentation and appearance of HTML elements on a webpage. This includes attributes like colors, fonts, spacing, layout, and other design-related properties.

Sheets

"Sheets" in CSS refers to the documents or files that contain the style rules. These can be in the form of external style sheets (separate files with a .css extension), internal style sheets (defined within the HTML document using the style tag), or inline styles (applied directly to HTML elements using the style attribute).
